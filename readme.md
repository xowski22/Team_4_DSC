## Prerequisites

- Python 3.10
- Jupyter Notebook

## How to run project

1. Clone the repository
2. Install the dependencies
3. Download the data from Kaggle:
    - `kaggle competitions download -c product-recommendation-challenge`
4. run the notebooks
